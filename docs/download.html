系统要求

  • 操作系统: Windows 10/11 (64位)
  • 处理器: 双核处理器，1.8GHz 或更高
  • 内存: 最低 4GB RAM
  • 硬盘空间: 最低 100MB 可用空间
  • 显示分辨率: 1366 x 768 或更高

若要使用高级功能（如OCR识别），建议配置更高的内存和处理器。

依赖项安装

要充分利用文智搜的所有功能，需安装以下依赖项

1. Tesseract OCR (用于图像和扫描PDF的文字识别)

Tesseract OCR是专业版文智搜用于从图像和扫描文档中提取文字的引擎。

安装步骤：

  1. 访问 Tesseract OCR 下载页面
  2. 下载最新版本的Windows安装包 (例如: tesseract-ocr-w64-setup-v5.3.0.20221222.exe)
  3. 运行安装程序，选择要安装的语言，确保至少选择：
    • 中文简体 (chi_sim)
    • 英语 (eng)
  4. 安装到默认位置 (C:\Program Files\Tesseract-OCR)
注意： 安装时请确保选择"将Tesseract添加到系统PATH"选项，这样文智搜可以自动找到Tesseract。

2. Poppler (用于PDF处理)

Poppler是专业版文智搜用于处理和转换PDF文件的库。

安装步骤：

  1. 访问 Poppler Windows 发布页面
  2. 下载最新版本的ZIP包 (例如: Release-24.05.0-0.zip)
  3. 解压缩文件到 C:\Program Files\poppler-24.05.0\
  4. 添加Poppler的bin目录到系统PATH:
    1. 右键点击"此电脑"，选择"属性"
    2. 点击"高级系统设置"
    3. 点击"环境变量"
    4. 在"系统变量"部分，找到并选择"Path"变量，然后点击"编辑"
    5. 点击"新建"，添加 C:\Program Files\poppler-24.05.0\Library\bin
    6. 点击"确定"保存更改
重要：安装完Poppler后，您需要重新启动电脑或注销后重新登录，以使PATH环境变量的更改生效。

3. 验证安装

安装上述依赖项后，您可以通过以下步骤验证安装是否成功：

验证Tesseract:

  1. 打开命令提示符 (cmd.exe)
  2. 输入以下命令并回车：
tesseract --version

如果安装正确，您应该看到类似以下的输出：

tesseract 5.3.0
libgif 5.2.1 : libjpeg 9d : libpng 1.6.37 : libtiff 4.4.0 : zlib 1.2.12 : libwebp 1.2.4

验证Poppler:

  1. 打开命令提示符 (cmd.exe)
  2. 输入以下命令并回车：
pdfinfo -v

如果安装正确，您应该看到类似以下的输出：

pdfinfo version 24.05.0

安装文智搜

完成依赖项安装后，安装文智搜

安装步骤

  1. 运行您下载的安装包 (文智搜-v1.0.0-setup.exe)
  2. 按照安装向导的提示完成安装
  3. 安装完成后，从桌面或开始菜单启动"文智搜"
  4. 首次启动时，您需要设置索引存储位置和添加要索引的文件夹
提示：如果您已安装了依赖项但文智搜无法识别它们，可以在设置中手动指定Tesseract和Poppler的路径。
