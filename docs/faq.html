常见问题解答 (FAQ)

我们收集了用户最常提问的问题，希望能帮助您更好地使用文智搜

安装与配置

如何安装文智搜？

文智搜的安装非常简单，只需下载并运行安装程序，按照向导的指引完成安装即可。详细步骤如下：

  1. 访问 下载页面 获取最新安装包。
  2. 运行下载的安装程序（通常命名为 DocumentSearchSetup-X.X.X.exe）
  3. 按照安装向导的提示完成安装
  4. 安装完成后，从开始菜单或桌面快捷方式启动应用程序

首次运行时，建议先前往"设置"配置索引目录，然后创建索引。

需要安装额外的依赖吗？

文智搜基本功能无需额外安装依赖，但特定功能需要以下外部组件：

  • Tesseract OCR (专业版功能): 用于识别图片或扫描版PDF中的文字。

详细的安装指南可在下载页面找到。

系统要求是什么？

文智搜的系统要求较为宽松：

  • 操作系统: Windows 10 / 11 (64位)
  • 处理器：任何现代处理器（建议双核或更高）
  • 内存：最低 4GB RAM（OCR功能建议 8GB 或更高）
  • 硬盘空间：程序本身约 50MB，索引大小取决于文档数量（约为源文件大小的 20-30%）
  • 额外要求：如需使用 OCR 功能，建议有更多内存和处理器资源

索引问题

如何创建索引？

创建索引的步骤如下：

  1. 打开文智搜
  2. 点击主界面上的"文件"菜单，选择"设置" > "索引设置"
  3. 在"要索引的文件夹"部分点击"添加目录"，选择您希望索引的文件夹
  4. 设置"索引文件存储位置"，默认为用户文档文件夹下的 DocumentSearchIndex 目录
  5. 根据需要调整其他设置（如OCR、超时等）
  6. 保存设置后，返回主界面点击"创建索引"按钮

索引过程会自动开始，期间您可以看到进度显示。索引完成后，即可开始搜索。

索引过程需要多长时间？

索引时间主要取决于以下因素：

  • 文件数量和大小
  • 文件类型（PDF 和需要 OCR 的文件处理较慢）
  • 计算机性能（特别是处理器速度和磁盘速度）
  • 是否启用 OCR（OCR 会显著增加处理时间）

作为参考，在中等配置的计算机上，不使用 OCR 索引 1000 个普通办公文档（Word、Excel 等）大约需要 5-10 分钟。如果启用 OCR 并处理大量 PDF 文件，可能需要数小时。

索引是增量式的，后续更新只会处理新增、修改或删除的文件，速度会更快。

我可以索引哪些类型的文件？

免费版支持的文件类型：

  • Microsoft Office 文档（.docx, .xlsx, .pptx）
  • 文本文件（.txt, .rtf）
  • 网页文件（.html, .htm）

专业版额外支持的文件类型：

  • PDF 文件（.pdf），包括 OCR 支持
  • Markdown 文件（.md）
  • 电子邮件文件（.eml, .msg）
  • 压缩包中的文件（.zip, .rar）
为什么有些文件被跳过没有索引？

文件可能因以下原因被跳过：

  • 许可证限制：PDF、Markdown、邮件等文件类型需要专业版许可证
  • 文件损坏：文件可能已损坏或格式不兼容
  • 文件太大：超过了配置的大小限制
  • 处理超时：文件处理时间超过了配置的超时限制
  • 加密或受保护：加密的文件无法处理
  • 权限问题：程序没有足够权限访问该文件

您可以通过点击主界面上的"查看跳过文件"按钮查看详细信息，了解每个被跳过文件的具体原因。

搜索功能

支持哪些高级搜索语法？

文智搜支持以下高级搜索语法：

  • AND / + : 查找必须包含所有关键词的结果 (例如: 报告 +项目 或 报告 AND 项目)
  • OR 操作符：使用 OR 表示。例如：预算 OR 财务
  • NOT 操作符：使用 NOT 排除词语。例如：报表 NOT 季度
  • 短语搜索：使用引号搜索完整短语。例如："年度财务报表"
  • 组合使用：可以组合多个操作符。例如："财务报表" AND (2022 OR 2023) NOT 草稿
  • 通配符（专业版）：* 代表任意多个字符，? 代表单个字符。例如：财务*报表 或 report-20??.xlsx
如何过滤搜索结果？

文智搜提供多种过滤选项：

  • 文件类型: 在搜索栏旁或过滤器区域选择特定的文件格式 (如 .docx, .pdf, .txt)。
  • 大小过滤：在"文件大小(KB)"中指定最小和最大值
  • 日期过滤：使用"修改日期"选择器指定日期范围
  • 排序：可按相关度、修改日期、文件大小或文件路径排序
  • 搜索范围：选择"全文"搜索文件内容，或选择"文件名"仅搜索文件名

许可证问题

免费版和专业版有什么区别？

免费版功能：

  • 常见格式内容索引与搜索: .docx, .xlsx, .pptx, .txt, .html, .rtf
  • 基础搜索语法: AND, OR, NOT, 短语搜索
  • 按文件大小、修改日期过滤
  • 最多 5 个源目录索引

专业版额外功能：

  • PDF 文件支持（包括 OCR）
  • Markdown 和电子邮件文件支持
  • 压缩包内容索引
  • 通配符搜索 (* 和 ?)
  • 无限制的源目录索引
  • 更详细的索引配置选项

完整的功能对比可以在购买页面查看。

如何激活专业版？

激活专业版许可证的步骤如下：

  1. 确保您已从官方渠道购买了专业版许可证密钥。
  2. 在文智搜中点击"设置"菜单，选择"许可证管理"
  3. 在弹出的对话框中，输入或粘贴您的许可证密钥。
  4. 点击"激活"按钮。

激活成功后，专业版功能将立即可用，无需重启应用程序。

可以在多台电脑上使用同一个许可证吗？

通常，标准许可证仅授权在单台或指定数量的设备上使用。如果您需要在新电脑上使用：

  1. (如果可能) 在旧电脑上停用或移除许可证。
  2. 在新计算机上安装文智搜
  3. 使用您的许可证密钥在新电脑上激活。

注意：每个许可证密钥同一时间只能在一台计算机上激活。如需在多台计算机上使用，请购买额外的许可证。

错误与故障排除

程序启动时出现"无法加载配置"错误怎么办？

此问题通常与配置文件损坏有关。解决步骤：

  1. 关闭文智搜
  2. 导航到用户目录下的配置文件夹：C:\Users\[用户名]\AppData\Roaming\YourOrganizationName\DocumentSearchToolPySide
  3. 重命名或删除配置文件（通常是 .ini 文件）
  4. 重新启动应用程序，这将创建新的默认配置文件

注意：此操作会重置所有设置，您需要重新配置索引目录等。

OCR 功能无法正常工作怎么办？

OCR 功能问题通常与 Tesseract OCR 安装有关。请检查：

  1. 确认您已安装 Tesseract OCR，并且版本兼容（建议 v5.0.0 或更高版本）
  2. 确认环境变量正确设置（Tesseract 的安装目录应在系统 PATH 中）
  3. 确认已安装中文语言包（如需处理中文文档）
  4. 重启应用程序尝试重新索引

详细的 Tesseract OCR 安装和配置指南可在下载页面找到。

索引过程中电脑很卡怎么办？

首次建立索引或索引大量文件时，可能会消耗较多系统资源。建议：

  • 在电脑空闲时进行首次索引。
  • 关闭文智搜，然后重新启动，有时可以解决临时资源占用问题。
  • 检查设置中是否有关于索引性能/线程数的调整选项（如果提供）。
  • 如果问题持续，可以尝试减少一次性索引的目录数量，分批进行。

如果问题持续存在，请收集错误日志并联系我们获取支持。
